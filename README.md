# lt-2023-05-08
## クローンの方法
Desktopにリポジトリをクローンする
```
cd 
cd Desktop 
git clone https://github.com/amatech-repo/lt-2023-05-08.git
cd lt-2023-05-08
```

## Webサイトを確認する
1. `vs code`の拡張機能でlive serverをインストールする
2. `index.html`をvscode上で開く
4. 右下の`Go live`を押す
5. 編集などが自動で反映される
